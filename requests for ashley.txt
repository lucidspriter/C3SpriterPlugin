# Requests for Ashley

We're porting the legacy Spriter plugin to Addon SDK v2. The plugin works, but
several runtime operations rely on probing multiple method names because we
couldn't find the documented SDK v2 equivalent. We'd like to replace these with
the correct single API call.

---

## 1. Accessing a plugin's own animation frames at runtime

We use `SetHasAnimations(true)` and load atlas PNGs into the plugin's animation
frames at import time (editor-side). At runtime we need to enumerate those frames
(legacy equivalent: `GetObjectClass().GetAnimations()[0].GetFrames()`), but we
can't find a documented SDK v2 runtime API for this.

Currently we fall back to loading atlas PNGs from Project Files and slicing via
SCON metadata, which works but means duplicating image data.

**Question:** Is there a supported SDK v2 runtime API to access a custom plugin's
animation frames / image infos, or is project-file loading the intended approach?

---

## 2. Undocumented runtime APIs we're probing by name

The plugin controls associated Sprite instances in non-self-draw mode. For each
operation below, we probe multiple names and use whichever exists. What are the
correct SDK v2 names?

| Operation | What we probe | Where used |
|-----------|--------------|------------|
| Set origin/hotspot | `SetOriginX`/`setOriginX`, `SetOriginY`/`setOriginY` | pivot application on child sprites |
| Invalidate bbox | `SetBboxChanged`/`setBboxChanged` | after moving child sprites |
| Enable/disable collision | `SetCollisionEnabled`/`setCollisionEnabled` | toggling collision boxes per frame |
| Enumerate instances | `instances()`, `getAllInstances()`, `GetInstances()` | finding paired Sprite instances |
| Object type name | `GetName()`, `.name` | diagnostic logging |
| Instance UID | `GetUID()`, `getUID()`, `getUid()`, `.uid` | diagnostic logging |
| Container siblings | `GetSiblings()`, `getSiblings()` | finding paired instance via container |
| Picked/paired instance | `getPickedInstances()`, `getFirstPickedInstance()`, `getPairedInstance()` | event-context instance resolution |
| Sprite frame index (read) | `_currentFrameIndex`, `currentFrameIndex`, `animationFrame` | reading current frame |
| Sprite frame index (write) | `_DoChangeAnimFrame()`, `_changeAnimFrameIndex`, `animationFrame` | setting frame to match Spriter pose |
| No-premultiply blend | `renderer.SetNoPremultiplyAlphaBlend`/`setNoPremultiplyAlphaBlend` | legacy "no premultiplied alpha" property |
